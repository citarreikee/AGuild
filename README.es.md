## 🪙 Pago

El pago se realiza directamente entre el cliente y el aventurero. Métodos comunes: cripto (USDC/ETH), Wise, Stripe.

El Gremio no retiene fondos. Somos una capa de coordinación — no una plataforma de pagos.
