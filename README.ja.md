## 支払い

支払いはクエスト投稿者と冒険者の間で直接行われます。主な方法：暗号通貨（USDC/ETH）、Wise、Stripe。

ギルドは資金を預かりません。私たちは調整レイヤーであり、決済プラットフォームではありません。
