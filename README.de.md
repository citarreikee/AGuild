## Bezahlung

Die Zahlung erfolgt direkt zwischen Quest-Geber und Abenteurer. Übliche Methoden: Krypto (USDC/ETH), Wise, Stripe.

Die Gilde verwahrt keine Gelder. Wir sind eine Koordinationsschicht — keine Zahlungsplattform.
