## Как это работает

<table>
<tr>
<td align="center" width="33%"><b>1. Подать</b><br><sub>Заказчик создаёт Issue — что нужно сделать, сколько платит. Вот и всё.</sub></td>
<td align="center" width="33%"><b>2. Взять</b><br><sub>Искатель комментирует, чтобы взять квест. Заказчик назначает его. Работа начинается.</sub></td>
<td align="center" width="33%"><b>3. Завершить</b><br><sub>Работа сдана. Issue закрыт. Оплата напрямую. Репутация заработана.</sub></td>
</tr>
</table>

Без комиссий платформы. Без посредников. Без сложностей.

## 🪙 Оплата

Оплата происходит напрямую между заказчиком и исполнителем. Обычные способы: криптовалюта (USDC/ETH), Wise, Stripe.

Гильдия не хранит средства. Мы — координационный слой, а не платёжная платформа.
