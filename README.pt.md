## Pagamento

O pagamento é feito diretamente entre o cliente e o aventureiro. Métodos comuns: cripto (USDC/ETH), Wise, Stripe.

A Guilda não retém fundos. Somos uma camada de coordenação — não uma plataforma de pagamento.
