## 🗺️ How It Works · 工作流程

```
  📜 Quest Posted       ⚔️ Adventurer Claims      🔨 Work Happens       ✅ Quest Complete       💰 Reward Paid
  📜 悬赏发布           ⚔️ 冒险家认领              🔨 执行任务           ✅ 验收完成             💰 赏金支付
```

<br>

<table>
<tr>
<td align="center" width="16%"><b>1. Post</b><br><sub>Quest Giver opens an Issue with the quest template — what needs doing, what it pays</sub></td>
<td align="center" width="16%"><b>2. Claim</b><br><sub>Adventurer comments <code>/claim</code> — stakes their intent, proposes approach</sub></td>
<td align="center" width="16%"><b>3. Build</b><br><sub>Quest Giver assigns them. Work begins. Communication stays in the open.</sub></td>
<td align="center" width="16%"><b>4. Deliver</b><br><sub>Adventurer opens a PR or links deliverables. Quest Giver reviews.</sub></td>
<td align="center" width="16%"><b>5. Complete</b><br><sub>Approved → <code>✅ Completed</code>. Reward paid. Reputation earned.</sub></td>
<td align="center" width="16%"><b>6. Rise</b><br><sub>Adventurer gains rank. Higher-tier quests unlock. The cycle continues.</sub></td>
</tr>
</table>

## ✨ Demo — A Quest from Start to Finish

### 📜 Quest Giver posts

> **Title:** Build a Telegram bot that summarizes YouTube videos
> **Type:** ⚔️ Code &nbsp;|&nbsp; **Reward:** $800 &nbsp;|&nbsp; **Difficulty:** ⭐⭐⭐ Veteran
> **Deliverables:** Working bot + deployed + README

### ⚔️ Adventurer claims

```
Adventurer ❯ /claim

             I can take this. Built 3 Telegram bots in the past year.
             Approach: Python + yt-dlp + OpenAI Whisper for transcription
             → GPT-4o for summarization → python-telegram-bot for interface.
             Delivery: 5 days. 2 rounds of revision included.
```

### ✅ Quest Giver accepts → Adventurer delivers → Complete

```
Quest Giver ❯ Delivered ahead of schedule. The Whisper pipeline handles
              multi-language beautifully. Marking complete. 💰 $800 sent.

Adventurer  ❯ Pleasure working with you. +1 quest to the guild card.
```

## ⚡ The Vision · 愿景

> *The factory, the office, the 9-to-5 — these are relics of an industrial age. The information age demanded new forms of work, and we got gig platforms that treat humans like interchangeable widgets.*
>
> *The AI age demands something different. When one exceptional person with AI can do the work of ten, that person isn't a "freelancer" — they're a **hero**. And heroes don't fill out timesheets. They take quests.*

> *工厂、办公室、朝九晚五——这些都是工业时代的遗物。信息时代要求新的工作形式，而我们得到的却是把人当作可替换零件的零工平台。*
>
> *AI 时代要求的完全不同。当一个卓越的人借助 AI 能完成十个人的工作时，他就不再是「自由职业者」——他是**英雄**。而英雄不填工时表。他们接悬赏。*
