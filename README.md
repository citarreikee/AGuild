## ⚔️ What is This? · 这是什么？

AI makes great people 3×, 10×, even 100× more productive. If you're one of them, you shouldn't be limited by a single employer's salary band. You should be able to take quests from anywhere, deliver excellence, and be compensated accordingly.

This is a **quest board** — a new social contract for work.

| Role | Does |
|------|------|
| 🏰 **Quest Giver** · 悬赏发布者 | Posts bounties for real work |
| ⚔️ **Adventurer** · 冒险家 | Claims quests, delivers results, earns gold |
| 🛡️ **Guild** · 公会 | Provides the board, rules, and reputation system |

No middlemen. No platforms taking 30%. Just work that gets done.

AI 让优秀的人生产效率提升十倍百倍。如果你是这样的人，你不应该被单一雇主的薪资区间所限制。你应该能从任何地方接任务、交付卓越成果，并得到匹配的回报。

这是一个**悬赏看板**——一种新的工作社会契约。

## 🗺️ How It Works · 工作流程

```
📜 Quest Posted  →  ⚔️ Adventurer Claims  →  🔨 Work Happens  →  ✅ Quest Complete  →  💰 Reward Paid
📜 悬赏发布      →  ⚔️ 冒险家认领        →  🔨 执行任务     →  ✅ 验收完成      →  💰 赏金支付
```

| Step | Action |
|------|--------|
| 1 | **Quest Giver** creates an [Issue](https://github.com/citarreikee/AGuild/issues/new/choose) using the Quest template |
| 2 | **Adventurer** comments `/claim` to stake their interest |
| 3 | **Quest Giver** assigns them; quest moves to `⚡ Claimed` |
| 4 | **Adventurer** delivers via Pull Request (or linked work) |
| 5 | **Quest Giver** reviews, closes → `✅ Completed` |
| 6 | 💰 Reward is paid (off-platform for now — see Payment below) |

## 🪙 Payment · 支付

In this MVP phase, payment happens **off-platform** between Quest Giver and Adventurer.

| Method | Best for |
|--------|----------|
| **Crypto** (USDC / ETH) | Global, fast, low fees |
| **Wire / Wise** | Larger amounts ($500+) |
| **Stripe / PayPal** | Convenience |

> ⚠️ **The Guild does not hold funds yet.** Smart-contract escrow is on the v2 roadmap. Until then, exercise normal freelance caution — milestone-based payments recommended.
