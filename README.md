## How It Works

```
📜 Post a Quest  →  ⚔️ Adventurer Claims  →  ✅ Complete & Pay
```

1. **Quest Giver** [posts a quest](https://github.com/citarreikee/AGuild/issues/new?template=quest.yml) — what needs doing, what it pays
2. **Adventurer** comments to claim it. Quest Giver assigns them.
3. Work gets done. Quest Giver closes the issue. Reward is paid.

That's it. No platform fees, no middlemen, no complexity.

## Payment

Payment happens directly between Quest Giver and Adventurer. Common methods: crypto (USDC/ETH), Wise, Stripe.

The Guild does not hold funds. This is a coordination layer, not a payment platform.
